## **1. 事業概要**

### **ビジネス名**
「Code & Copy Forge」（コアランゲージハブ）

### **ビジネスコンセプト**
自社商品を開発し始めた起業家・マーケティング学習者に対して、11フレームワーク（リーンキャンバス）を使って「事業の言語化」を支援し、その成果をWebメディア×コピーライティング×内製化プロジェクトで受注するビジネス。

### **提供価値**
- **記事LP 18本**：11フレームワークの無料学習教材
- **パーソナル分析アプリ**：顧客の事業を診断
- **パーソナライズメルマガ**：顧客専用のガイダンス
- **物理レポート**：顧客の「オリジナルリーンキャンバス」
- **バックエンド受注**：Webメディア＆内製化プロジェクト

### **マネタイズ**
- **フロントエンド**：無料（記事LP、アプリ、メルマガ）
- **バックエンド**：月額 50-200万円の受注プロジェクト

## **5. パーソナル分析アプリ仕様**

### **アプリ概要**

| 項目 | 仕様 |
|------|------|
| **名称** | 「ビジネス言語化診断アプリ」 |
| **形式** | Web アプリケーション（No アプリDL） |
| **所要時間** | 15分 |
| **ヒアリング項目** | 29項目 |
| **対応フレームワーク** | 11フレームワーク完全対応 |
| **出力** | 回答データ → LLM → メルマガ + レポート自動生成 |

### **ヒアリング項目 29項目（詳細）**

#### **【1】PRB - 課題と前提条件（3問）**

**Q1. あなたの事業は何ですか？**
- 形式：自由記述（テキスト）
- 最大文字数：500字
- 例：「BtoB営業支援ツール開発」

**Q2. 今、最も困っていることは何ですか？（HARM視点）**
- 形式：複数選択 + 自由記述
- 選択肢：
  - □ H（健康）：身体的課題
  - □ A（野心）：キャリア・成長欲
  - □ R（人間関係）：信頼・ネットワーク
  - □ M（お金）：経済的課題
- 記述：詳細（200字）

**Q3. その課題の前提条件は？**
- 形式：自由記述
- 最大文字数：300字
- ヒント例：「新入社員だから」「急成長中だから」

#### **【2】EMP - ペルソナとカスタマージャーニー（4問）**

**Q4. あなたのメインターゲット（理想の顧客像）は？**
- 形式：自由記述
- 最大文字数：200字
- 例：「35-50歳の営業責任者、年商1-10億円企業」

**Q5. その顧客は、今どんな状態ですか？（共感マップ）**
- 形式：複数フィールド
  - 見ているもの：テキスト
  - 聞いていること：テキスト
  - 考えていること：テキスト
  - 痛み：テキスト
  - 得たいもの：テキスト
- 最大文字数：各100字

**Q6. その顧客の『現在地』から『理想』までのジャーニーは？**
- 形式：テキスト（ストーリー形式）
- 最大文字数：500字
- 例：「案件化率が低い → 提案資料作成に時間がかかってる → 実はターゲット定義がない」

**Q7. 複数のターゲット層があれば、それぞれ記述**
- 形式：オプション記述
- 最大文字数：300字

#### **【3】BNF - ペルソナの理想の姿（2問）**

**Q8. その顧客が、あなたの商品を使ったら、6ヶ月後どうなってますか？**
- 形式：自由記述
- 最大文字数：300字
- 例：「営業案件化率が30% → 50%に上がった」

**Q9. 1年後は？3年後は？**
- 形式：自由記述（段階的）
- 最大文字数：各200字

#### **【4】SLT - 解決策（2問）**

**Q10. あなたが提供する商品・サービスは何ですか？**
- 形式：自由記述
- 最大文字数：400字
- 記述要素：商品概要、内容、提供フロー

**Q11. それは、ターゲットの課題をどう解決しますか？**
- 形式：自由記述
- 最大文字数：300字
- 記述要素：具体的メカニズム

#### **【5】MKT - マーケティングファネル（2問）**

**Q12. 現在、あなたはどこでターゲットと接点を持ってますか？**
- 形式：複数選択
- 選択肢：
  - □ Webサイト
  - □ SNS
  - □ 営業（直接接触）
  - □ イベント
  - □ 紹介
  - □ その他

**Q13. 次の3ヶ月で、最優先にやることは何ですか？**
- 形式：単一選択
- 選択肢：
  - □ 認知を増やす

  - □ 比較検討を支援
  - □ 購買を促す
